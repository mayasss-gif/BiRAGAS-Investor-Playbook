# BiRAGAS · Investor Playbook

A coaching tool for pitching BiRAGAS. Single-page React app with audio narration on every section. Same architecture as [`BiRAGAS-Investor-Pitch-Guide-v2`](https://mayasss-gif.github.io/BiRAGAS-Investor-Pitch-Guide-v2/) — built from the content of [`BiRAGAS-Investor-Walkthrough`](https://mayasss-gif.github.io/BiRAGAS-Investor-Walkthrough/).

**Live:** https://mayasss-gif.github.io/BiRAGAS-Investor-Playbook/

## What's inside

### Pitch Arc (10 sections)

1. **The Hook** — open with the wound, not the solution
2. **The Problem** — correlation is not a business strategy
3. **The Wedge** — we identify the driver, not the bystander
4. **The Platform** — raw data in, causal map out
5. **The Engine** — Layer-3 causality, what sets us apart
6. **The Market** — three verticals, $50B+ addressable
7. **The Moat** — foundation gap, not feature gap
8. **The Model** — modular licensing, CLIA, companion diagnostics
9. **The Ask** — SAFE · 20% disc · $40M cap · Refund Right
10. **The Bet** — why now, why us

### Objections tab (8 Q&A)

Pre-cooked answers for the eight hardest questions investors ask.

### Audio

`[🔊 ON / 🔇 OFF]` toggle in the header. When ON, each section / objection plays an Ava Neural narration coaching the pitcher on tone and tactics.

## Built with

Pure HTML + React 18 UMD + Babel Standalone (CDN). No build step.
